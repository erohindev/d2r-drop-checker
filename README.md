# D2R Drop Checker

Customizable drop notifier

# How to use

1. Unpack zip
2. Right click `exocetblizzardot-medium.otf` -> **INSTALL FOR ALL USERS** (it will copy the font to Windows/Fonts/)
3. Edit config `filters/filters.txt`
4. Run game
5. Run `DC.exe`
